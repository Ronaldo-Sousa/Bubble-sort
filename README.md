# Bubble-sort
Demonstrando a implementação do conceito de ordenação de dados seguindo o método "bubble sort". O método recebe esse nome porque funciona fazendo os elementos do vetor
lentamente "flutuarem" para a posição certa como se fossem bolhas, é um algoritmo relativamente simples e curto, mas não é eficiente para grandes quantidades de dados, nesse
contexto "grande" seriam milhões. O código começa criando a função bubbleSort, que recebe como parâmetros um vetor (que é o que vai ser ordenado) e um número inteiro que é o tamanho desse vetor, ao abrir a chave da função nós declaramos três variáveis do tipo inteiro, duas serão contadores e uma variável auxiliar, então nós criamos duas estruturas "for" uma dentro da outra, a primeira usa o contador "i" (não é necessário, mas é uma convenção que seja) inicializado em 0 e percorre o vetor uma quantidade de vezes igual ao tamanho do vetor menos 1. Então dentro desse for nós criamos um outro que usa o contador "j" (de novo, não é obrigatório, mas é uma convenção) inicializado em 0 e percorrendo o vetor uma quantidade de vezes igual ao tamanho do vetor menos "i" e menos 1, é importante usar os dois laços "for" porque isso garante que o vetor inteiro será percorrido e que o processo será repetido quantas vezes forem necessárias para o vetor estar totalmente ordenado. Dentro do segundo "for" nós usamos uma estrutura de seleção "if", sendo que a condição é o vetor[j] ser maior que o vetor[j] mais 1, ou seja, o elemento do vetor ser maior que o elemento imediatamente à sua direita, se essa condição for verdadeira, então
a variável auxiliar recebe o valor do vetor[j], o vetor[j] por sua vez recebe o valor de vetor[j] mais 1, e por fim o vetor[j] mais 1 recebe o valor da variável auxiliar, é
assim que é feita a troca dos elementos, um é comparado com o da direita, e se for maior eles trocam de posição. Feita a função, vamos para a "main" do código, em que criamos o nosso vetor com o tipo, o nome e os seus elementos, depois criamos uma variável de tipo inteiro para dizer o tamanho do vetor (isso é necessário para os laços de repetição funcionarem) e aí invocamos a função bubbleSort passando como parâmetros o nome do vetor que criamos e o nome da variável que criamos para armazenar o tamanho dele, e a função faz a ordenação. Feita a ordenação, podemos imprimir o vetor ordenado ao final pra ver se funcionou. Vale ressaltar que nesse algoritmo o vetor é organizado em ordem crescente, mas para mudar para a ordem decrescente é bem simples, basta substituir o símbolo ">" dentro do if para "<", aí você inverte a lógica da comparação no código inteiro.
